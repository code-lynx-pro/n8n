# n8n Workflow Repository
This repository contains exported workflow(s) from n8n.io. For backup, sharing, and version control. 

## Workflow Summary
This workflow automatically fetches trending repositories from GitHub, extracts and processes repository data, appends results to a sheet, and can send notifications. It also uses AI for deeper insights and text processing. Credentials are NOT included and must be re-added after import.

## How to use
- Download the JSON file(s).
- Import into your n8n instance via the Import menu.
